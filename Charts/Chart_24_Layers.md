## Quick Reference: Layer Combinations

When testing, you can combine layers:
- "Is it in a **time** layer?" → Go to A1
- "Is it in a **body/being** layer?" → Go to A2/A3
- "Is it in an **energy structure**?" → Go to B1
- "Is it in a **mind** layer?" → Go to B2
- "Is it **dimensional**?" → Go to B3
- "Is it from **origin/conception**?" → Go to C1
- "Is it **hidden/concealed**?" → Go to C2
